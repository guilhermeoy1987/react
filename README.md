# Iniciando com React - EBAC

Este repositório contém os primeiros passos e exercícios práticos utilizando a biblioteca React para a criação de interfaces modernas e componentizadas.

## O que foi desenvolvido:
* **Criação de Componentes:** Estruturação da interface em blocos reutilizáveis.
* **Uso de JSX:** Implementação de marcação estrutural integrada ao JavaScript.
* **Gerenciamento de Props:** Prática de passagem de dados entre diferentes partes da aplicação.
* **Hooks Iniciais:** Introdução ao controle de estado de componentes.

## Tecnologias:
* React.js
* JavaScript (ES6+)
* HTML5 / CSS3
* Git / GitHub
